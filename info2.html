Tic Tac Code was the game selected to undertake of the Grade 12 ICT Students in Arellano University Andres Bonifacio Campus.

Tic Tac code game is a game in which two participants alternately draw an O or an X.

If you are the first to draw the line linking x or o, you win this game.

You have to make the right decision in this mental sport so that your opponent can't draw a straight line and win.

There is a chance of a draw if you don't outplay your opponent right away.
